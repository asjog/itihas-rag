ठेविलें. या नवीन कामानें शिवाजीस जो अनुभव आला, तो पुढें त्याच्या उपयोगीं पडला.
शिवाजीचें हें शिक्षण चालू असतां त्यानें कांहीं गोष्टी विशेष लक्षांत ठेविल्या. ( १ ) लोकांच्या जमिनीवर सरकार देणें कसें लागू करावें; ( २ ) साऱ्याची ही रक्कम कशी वसूल करावी; ( ३ ) राज्य चालतें कसें व सरकार आपला कारभार चालवितें कसें; ( ४ ) आणि लोकांचे तंट्यांचा न्याय कसा तोडावा. ह्या गोष्टींचा अनुभव घेतल्यावर त्यास असें कळून आलें कीं रयत लोकांस शेतीवाडींत आनंद वाटला पाहिजे, आणि प्रत्येक गांवास आपला अंत:स्थ कारभार करण्याची संपूर्ण मोकळीक पाहिजे. येणेंकरून सरकारची जबाबदारी कमी होऊन लोकही संतुष्ट राहतात. हेंच धोरण शिवाजीच्या पुढें उपयोगीं पडले. प्रस्तुत जहागिरींत चार प्रकाराची सुधारणा करण्याची अत्यंत जरूर होती:-
( १ ) प्रांताच्या बंदोबस्तासाठीं लष्कर ठेवणें; ( २ ) वाघ, लांडगे वगैरे जनावरांपासून शेतीस व लोकांस होणारा उपद्रव बंद करणें, ( ३ ) उत्पन्न वाढविण्यासाठीं जमीन महसूल ठरवून व लोकांस अनेक सवलती देऊन शेती सुधारणें आणि ( ४ ) तंटे तोडून लोकांस उद्योगास लावणें.
( १ ) लष्कर. – शहाजीची मूळची जहागीर पुणे व सुपे हे प्रांत असून पुढें लवकरच कर्नाटकांतील मोहीम फत्ते केल्याबद्दल बारामती, इंदापूर व मावळ हे प्रांत त्यास मिळाले. मावळे लोक दिसण्यांत अडाणी पण प्रामाणिक, मेहनती व चपळ होते. ह्या लोकांवर शिवाजीची भक्ति विशेष बसली. दादाजीनें देशाच्या बंदोबस्ताकडे त्यांचा चांगला उपयोग केला. बिनकवाइती लोकांची एक पायदळ पलटण त्यानें तयार केली; आणि तिजवरील हुकमतीचे दर्जे ठरवून दिले. तसेंच गांवोगांव चौक्या पहारे बसवून, चोरांची भीति नाहींशीं केली; आणि आपल्या हद्दींतील किल्ले वगैरे दुरुस्त करून त्यांजवर शिबंदी ठेवून दिली,
( २ ) दादाजीनें लोकांस असें जाहीर केलें, कीं जो कोणी वाघ किंवा लांडगा मारून आणील, त्यास बक्षीस मिळेल. बक्षिसाच्या आशेवर पुष्कळ जनावरें मारलीं गेलीं व देशांतील उपद्रव नाहींसा झाला.
( ३ ) शेतीच्या कामीं दादाजीनें पुष्कळ उपाय योजले. दरएक गांवीं फिरून, मलिकंबराचे पद्धतीप्रमाणें जमिनीची मोजणी व प्रतबंदी करून
