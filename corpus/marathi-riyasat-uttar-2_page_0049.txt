सिंधांची आपली मानमहत्त्वें वांटणी व्हावी. आम्ही ज्येष्ठ, तो ज्येष्ठत्वाचा मान आम्हांकडे आहे तसा चालवावा, म्हणजे देशीं येणें घडेल. ’ याचा अर्थ ज्येष्ठत्वाचा मान मात्र पाहिजे, पण ज्येष्ठत्वाचे कष्ट किंवा जबाबदारी नको. हिंगण्यानें बाईस कळविलें, ‘ नानांचें व तात्यांचें लक्ष तुमचे ठायीं पूर्वीपासून आहे तें अधिकोत्तर होईल. सिंद्यापाशीं तुमचा परिणाम लागणार नाहीं. पूर्वी ताराबाई (भोसले) व उमाबाई (दाभाडे) आदिकरून बायकांस कारभाराचें यश नाहीं. तुम्ही धर्म करितां; पदरीं पुण्यसामुग्री, महेश्वरी रेवातिरीं सत्क्षेत्र धरून पक्क्या रीतीनें आहां, त्यास बट्टा लागेल.’ ‘बाई सर्वथैव सरकारचे लक्षांत.जें नाना तात्यांचें लक्ष व सरकारमर्जी तेंच करावें.’ तुकोजीबावा म्हणतात, “बाईपुढें आमचा उपाय चालत नाहीं. बाईची सत्ता आम्हांवर नसावी. एक हातीं असावें. बाईचें लक्ष सर्वात्मना आपले उभयतांचे मर्जीवर. त्यांचे तोंडीं उत्तरें निघतात कीं, ‘मजला यांचे सरदारीशीं दरकार नाहीं. राहिले आयुष्यांत माझें महत्त्व आहे, त्यास धक्का, श्रीमंतांचे पायीं माझें आचरण शुद्ध व मल्हारजीबाबांचें पुण्य आहे तर, लागणार नाहीं; व जीवितव्य आहे तों लागूं देणार नाहीं. हृदयस्थ आहे तोच बुद्धीस प्रेरक होऊन यांतही सहस्र तोडी काढील.’ अशा प्रकारची हीं हिंगण्याचीं पत्रें होळकरशाहीची अंतःस्थिति समजण्यास फारच उपयोगी आहेत. येथें फक्त थोडेसे उतारे घेतले आहेत. भाषेच्या दृष्टीनेही हा भाग मार्मिक आहे.

हीं पत्रें पावल्यावर नानानें पुढील व्यवस्थेचीं कलमें लिहून पाठविलीं, तीं अशीं:—
( १ ) दहा हजार फौज बाळगून सांगूं तेथें मोहीमेवर जावें. पैका अहल्याबाईचे विचारें सरंजामांतून घ्यावा.
( २ ) बाईचे महत्त्वास् अंतर करूं नये.
( ३ ) तुकोजींनीं अखत्यारीनें कारभार करावा.
( ४ ) सर्वांनीं कारभाऱ्यांचें आज्ञेंत चालावें.
( ५ ) कै. मल्हाररावांचे वेळचें जुने माणूस काढू नये.
( ६ ) सरदारीचा गळाठा, सबब सरकारांतून समागमें देऊं त्याचे दाखल्यानें कलमांची वहिवाट करावी.
( ७ ) सरकार देण्याचा निकाल सोयीसोयीनें करावा.
