बाळाजी विश्वनाथाचा तह आंगऱ्यास प्रत्यक्ष व अप्रत्यक्ष फायदेशीर झाला. प्रत्यक्ष त्यास स्वतंत्र मुलूख तोडून मिळाला, त्याशिवाय बाहेरच्या भानगडींतून त्याची मुक्तता होऊन स्वस्थता प्राप्त झाल्यामुळें कोंकणांत आपली सत्ता मजबूत करण्यास त्यास सवड मिळाली. कान्होजी शूर व हुशार होता. पुढील पंधरा वर्षांत त्यानें आपली शक्ति पुष्कळच वाढविली. सन १७१५ त ‘ वसईपासून सावंतवाडीपर्यंत पूर्ण स्वतंत्र किंवा अंशतःपरतंत्र असे अधिकारी येणेंप्रमाणें होते. ( १ ) वसई, ठाणें व चेऊल या ठिकाणीं पोर्तुगीझ लोक; ( २ ) मुंबईस इंग्रज; ( ३ ) जंजिऱ्यास हबशी; ( ४ ) अलिबागेस आंगरे; ( ५ ) सावंतवाडीस सावंत; ( ६ ) मालवण पासून पन्हाळ्यापर्यंतच्या प्रदेशांत कोल्हापूरकर; व ( ७ ) गोव्यास पोर्तुगीझ. आंगऱ्याच्या व्यतिरिक्त स्वतंत्र असे पांच सत्ताधारी ह्या वेळीं कोंकणांतील निरनिराळ्या भागांत असल्यामुळें शाहूच्या वतीनें आंगऱ्यास वेळोवेळीं ह्या पांच लोकांशीं युद्धप्रसंग पडत. ह्या युद्धप्रसंगांत आंगऱ्याला आपलें लक्ष स्वसत्ता बळकट करण्याकडे लावतां आलें, हा बाळाजी विश्वनाथाच्या तहाचा परिणाम होय. आरबी समुद्राच्या संबंध किनाऱ्यावर आपला ताबा असावा अशी आंगऱ्यांची इच्छा असल्यामुळें, लवकरच त्यांचा झगडा इंग्रजांशीं सुरू झाला. कारण इंग्रजही दर्यावर्दी कामांत ह्या वेळेस पुढें सरसावू लागल्यामुळें आंगऱ्यांचे प्रतिस्पर्धी बनले. कान्होजीची मोठी कामगिरी ह्मणजे त्यानें इंग्रजांशीं चालविलेला हा अपूर्व झगडा होय. इंग्रजांच्या हकीकतींत आंगऱ्यांस त्यांनीं चांचे ह्मणून संबोधून, त्यांनीं त्यांजवर दांत ओठ खाऊन पाखडलेली आग वाचली ह्मणजे आंगऱ्यांच्या कामगिरीची कल्पना होते.

याचा एक मासला येथें देतों. ‘ दर्यावर लूट करणाऱ्या या सरदाराची हांव विजयदुर्ग हस्तगत करूनच थांबली नाहीं. पोर्तुगीज व इतर व्यापाऱ्यांस घालवून देऊन पश्चिम किनाऱ्यावर बंदरें व किल्ले बांधून तो एक स्वतंत्र राजाच बनला. एकदां सुंदर आरबी घोड्यांनीं भरलेलें जहाज सहज गल्या त्याच्या हस्त-
