( ४ ) दिल्लीजवळचा कांहीं प्रदेश निजामाचा समजावा.
( ५ ) निजाम पेशव्यांमधील वादाचे मुद्दे एकमेकांनीं सोडून द्यावे आणि निजामानें दरसाल चौथाईची रक्कम देत जावी.
( ६ ) बेदरसुभा हें पेशव्याचें खासगी वतन समजले जातें तें सोडून देण्याची शिफारस नानानें बाजीरावास करावी.
( ७ ) निजामलीचे खर्चास नानानें दोन लाख रु. द्यावे.
( ८ ) निजामाचे मुलखावर टिपूनें चाल केल्यास, नानानें इंग्रजांची मदत आणून बंदोबस्त करावा.
( ९ ) निजामाकडून पळून आलेले इसम पेशव्यानें परत हवालीं करावे.
( १० ) हा करार नानानें बाजीरावाकडून मंजूर करून द्यावा.
या महाडच्या तहाचा उल्लेख वसईचे तहांन असून दोहोंचें पर्यवसान एकच होतें.
“ सिंद्याचे कारभारी बाळाजी अनंत वगैरेनीं गैर रीत करून परशराम रामचंद्र यांस आपलें संमतांत घेऊन बाजीरावांस अटकेंत ठेविलें आहे, त्यास निजामसरकारची फौज, पेशव्यांची हुजरात, व रघूजी भोसले यांनीं एक होऊन बाजीरावांस एक जागां करावें. अजमुलू उमरावांनीं जंगी सरंजामानें मसलतीस शरीक असावें. कंपनी इंग्रज बहादुर या मसलतीस शरीक होतील. निदान शरीक न जाल्यास प्रतिकूलता तरी त्यांजकडून व्हावयाची नाहीं, याची मजबुदी दोस्तदारानें (=नानानें ) केली असे. निजाम सरकारचा मुलूक, दौलताबादचा किल्ला वगैरे व नगदी ऐवज एकंदर देण्याचा करार खर्ढ्याचे मुक्कामीं जाला तो समग्र मुलूक व किल्ला व नगदी ऐवजाचे दस्तैवज माघारे देऊं; व पेसजी यादगीरचे मुक्कामीं स. १७८५ त तहनामा जाला त्याप्रमाणें पुढें चालावें. टिपूची फौज नमूद होऊन आल्यावर त्याचा बंदोबस्त इंग्रजांकडून व निजामाकडून व्हावा. या करारावर श्रीमंतांचे हातचें दस्तक करून देऊं. दोनही दौलतींच्या दरम्यान बिघाड कोणी कितीही समाजाविल्यास मनांत येऊं नये. वरचेवर स्वच्छता करीत जावी. कोणी बखेडखोर उभा राहिला तर दोघांनीं एक होऊन निराकरण करावें. एकूण कलमें सुमारें चौदा. ”
हा करार गोविंदराव पिंगळ्यानें कसा घडवून आणिला त्याचें वर्णन नागपुर बखरींत आहे. या कागदावरून नानाच्या कारस्थानाची स्पष्टता होते. इंग्रजानें नानास आपल्या बाजूस वळविलें होतें ही गोष्ट त्यांतच नमूद आहे. टिपू व रघूजी भोसले यांजकडेही नानानें असेंच संधान जमविलें. वस्तुतः या करारानें कोणाचा फायदा
